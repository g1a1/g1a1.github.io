Some of my Projects




Game (Unreal Engine)

Designed and prototyped ideas for a basic UDK game. After finalizing the design, we started work on various components of the game. The primary areas of focus included map design, player camera, user controls, weapons, and enemy AI. As you can see from the screenshot, we had as much fun playing our game as making it.



Java Client Server

Used RMI and JDBC to access information from a MySQL database. The RMI Server sends the requested data to the RMI Client. After receiving the data, it is processed to be displayed in chart form using the JFreeChart library.



TCP/IP Connection

Worked as a team to implement TCP/IP data communications between a server application and a client Arduino controlled Robot. The client received commands through network packets and successfully executed given instructions. These commands included status reports, directions/ duration for the robot to move, sleep, and closing the connection.
